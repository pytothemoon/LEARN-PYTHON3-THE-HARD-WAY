### 29. if문

Ex29.py
```
사람 = 20
고양이 = 30
개 = 15


if 사람 < 고양이:
    print("고양이가 너무 많아요! 세상은 멸명합니다.")

if 사람 > 고양이:
    print("고양이가 많지 않아요! 세상은 지속됩니다.")

if 사람 > 개:
    print("세상은 침에 젖습니다!")

if 사람 < 개:
    print("세상은 말랐습니다!")


개 +=5

if 사람 >= 개:
    print("사람은 개보다 많거나 같습니다.")

if 사람 <= 개:
    print("사람은 개보다 적거나 같습니다.")


if 사람 == 개:
    print("사람은 개입니다.")
```

실행결과
```
고양이가 너무 많아요! 세상은 멸명합니다.
세상은 침에 젖습니다!
사람은 개보다 많거나 같습니다.
사람은 개보다 적거나 같습니다.
사람은 개입니다.
```

### 30. else와 if

Ex30.py
```
사람 = 30
차 = 40
트럭 = 15

if 차 > 사람:
    print("차를 타야 해요.")
if 차 < 사람:
    print("차를 안 타야 해요.")
else:
    print("결정할 수 없어요.")

if 트럭 > 차:
    print("트럭이 너무 많아요.")
elif 트럭 < 차:
    print("트럭을 탈 수도 있어요")
else:
    print("아직도 결정할 수 없어요.")

if 사람 > 트럭:
    print("좋아요 트럭을 탑시다.")

else:
    print("좋아요 그럼 집에 있죠.")
```

실행결과

```
차를 타야 해요.
결정할 수 없어요.
트럭을 탈 수도 있어요
좋아요 트럭을 탑시다.
```

Ex31.py
```
print("""문이 두 개 있는 어두운 방에 들어왔습니다.
1번과 2번 중 어느 방으로 돌아갈까요 ?""")

문 = input("> ")

if 문 == "1":
    print("거대 곰이 치즈 케이크를 먹고 있습니다.")
    print("무엇을 할까요?")
    print("1. 케이크를 뺏는다.")
    print("2. 곰에게 소리를 지른다.")

    곰 = input("> ")

    if 곰 == "1":
        print("곰이 당신의 머리를 먹어치웁니다. 잘 했어요!")
    elif 곰 == "2":
        print("곰이 당신의 다리를 먹어치웁니다. 잘 했어요!")
    else:
        print(f"음, {곰} 행동을 하는 것이 더 나았나보네요.")
        print("곰이 도망갑니다.")

elif 문 == "2":
      print("당신은 크툴루 눈동자의 끝없는 심연을 처다봅시다.")
      print("1. 블루베리.")
      print("2. 노란 재킷 발래집게.")
      print("3. 권총이 울부짖는 가락 이해햐기")

      광기 = input(">")

      if 광기 == "1" or 광기 == "2":
          print("당신의 육체는 젤리푸딩의 마음의 힘으로 살아남습니다.")
          print("잘 했어요!")

      else:
          print("광기가 당신의 눈을 썩어 문드러진 시궁창으로 만듭니다.")
          print("잘 했어요!")

else:
    print("비틀거리다 발을 헛디뎌 칼날로 떨어져 죽습니다. 잘 했어요!")
```
