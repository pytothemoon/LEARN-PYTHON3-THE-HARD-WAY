### 21. 반환하는 함수
```
def 더하기(a,b):
    print(f"덧셈 {a}+{b}")
    return a+b

def 빼기(a,b):
    print(f"빼기 {a}-{b}")
    return a-b
    
def 곱하기(a,b):
    print(f"곱하기 {a}*{b}")
    return a*b
    
def 나누기(a,b):
    print(f"나눗셈 {a}/{b}")
    return a/b

print("함수만으로 계산해봅시다.")

나이=더하기(30,5)
키=빼기(78,5)
몸무게=곱하기(90,2)
```
